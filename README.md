# 📄 AI Resume Analyzer (ATS Optimizer)

An AI-powered web application that analyzes resumes and provides ATS-based scoring and feedback.

## 🚀 Features
- Extracts text from PDF resumes
- Calculates ATS score based on keyword matching
- Identifies important technical keywords
- Provides AI-like suggestions for improvement

## 🛠️ Tech Stack
- Python
- Streamlit
- NLP (Regex-based parsing)
- PyPDF2

## 📦 Installation

```bash
git clone https://github.com/yourusername/resume-analyzer.git
cd resume-analyzer
pip install -r requirements.txt
streamlit run app.py
```
